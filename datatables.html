Datatables

This page contains an example of the JQuery Datatables plugin. I've used this in a few projects and it's really handy for transforming a straightforward HTML table into something a bit more dynamic. I've dropped a quick example in below, but check out the Datatables website to find out more!

Mix this in with some dynamic server side code to populate the tables and you can get a nice dashboarding app up and running in no time!
